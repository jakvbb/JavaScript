Zadanie 41 - Pętle w JavaScript

1. Pętla FOR

Nazwa: for

Konstrukcja:

for (var i = 0; i < 5; i++) {
}

Opis: Pętla wykonuje kod określoną ilość razy. Warunek sprawdzany jest przed każdą iteracją.

Schemat blokowy: Start → Inicjalizacja → Warunek → Kod → Zwiększanie → (jeśli warunek spełniony to powrót do kodu, inaczej Stop)

2. Pętla WHILE (kontrolowane wejście)

Nazwa: while

Konstrukcja:

while (warunek) {
}

Opis: Pętla wykonuje kod dopóki warunek jest prawdziwy. Sprawdzany jest na początku.

Schemat blokowy: Start → Warunek → Kod → (jeśli warunek spełniony to powrót do warunku, inaczej Stop)

3. Pętla DO...WHILE (kontrolowane wyjście)

Nazwa: do...while

Konstrukcja:

do {
} while (warunek);

Opis: Pętla wykona kod przynajmniej raz, ponieważ warunek sprawdzany jest na końcu.

Schemat blokowy: Start → Kod → Warunek → (jeśli warunek spełniony to powrót do kodu, inaczej Stop)
